Tender Buttons — a philosophical lexicon presentation
=====================================================

TO VIEW: double-click "index.html" (it opens in your default browser).

This is for testing only, but it stands fully alone and can be run locally 
— no installation, no internet connection, no server required.
Everything (the full text of Gertrude Stein's "Tender Buttons", the 54-term
lexicon, and one image per term) is embedded in the files in this folder.

The presentation steps through the lexicon automatically (about one term every
5 seconds). Controls are on the right; you can Play/Pause, step Prev/Next, and
adjust the look. Keyboard: Space = play/pause, ← / → = previous / next term.  
Pressing "H" hides and shows the control panel.

Keep all three files together in the same folder:
  index.html   the page you open
  app.js       the program
  data.js      the embedded text + images

If your browser shows a blank page, try a different browser (Chrome, Edge,
Firefox, or Safari should all work).

Note that this package is assembled automatically by script, so feel free to 
make any changes here, but that they are only applicable to this package
and will not propagate to any subsequent builds without intervention by
George.
